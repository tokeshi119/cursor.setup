# Custom Commands

このファイルには、エージェントが実行できるカスタムコマンドが定義されています。コマンドは `/` でトリガーされます。

## `/pr`

現在の変更に対してプルリクエストを作成します。

1. `git diff`でステージされた変更とステージされていない変更を確認する
2. 変更内容に基づいて明確なコミットメッセージを書く
3. 現在のブランチにコミットしてプッシュする
4. `gh pr create`を使用してタイトルと説明でプルリクエストを開く
5. 完了したらPR URLを返す

## `/review`

コードレビューを実行します。

1. Linterを実行する
2. よくある問題をチェックする
3. 型チェックを実行する
4. テストを実行する
5. 注意が必要な点を要約する

## `/test`

コミット前のテストを実行し、失敗した場合は修正を試みます。

1. `git diff`で変更内容を確認する
2. `npm run test`を実行する
3. テストが失敗した場合、エラーを分析する
4. 必要に応じて修正を加える
5. すべてのテストが通るまで反復する
6. すべてのテストが通ったことを確認し、コミット準備が整ったことを報告する
