# コードベースの理解

このファイルには、コードベースを理解するためのガイドが定義されています。

## エージェントへの質問例

新しいコードベースに取り組む際は、以下のような質問をエージェントに投げかけることができます：

- 「このプロジェクトではログ出力はどのように実装されていますか？」
- 「新しいAPIエンドポイントを追加するにはどうすればよいですか？」
- 「このコンポーネントはどのようなエッジケースを扱っていますか？」
- 「なぜこの部分で`createUser()`ではなく`setUser()`を呼び出しているのですか？」

## 検索方法

エージェントは`grep`とセマンティック検索の両方を使ってコードベースを検索し、答えを見つけます。これにより、初めて触れるコードを素早くキャッチアップできます。

## コードパターンの理解

既存のコードパターンを理解するために、エージェントに以下のような質問をすることができます：

- 「このプロジェクトのコンポーネント構造はどのようになっていますか？」
- 「このプロジェクトで使用されているデザインパターンは何ですか？」
- 「このプロジェクトのテストの書き方はどのようになっていますか？」
